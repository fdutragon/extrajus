## 🔮 Funcionalidades de Poder (Core Capabilities)

### 1. ✨ Edição IA Cirúrgica
Diferente das IAs comuns que reescrevem documentos inteiros e quebram a formatação, o nosso motor opera no nível de **micro-cirurgiões**. Ele detecta, isola e reescreve parágrafos pontuais de forma totalmente localizada, mantendo a integridade estrutural, a numeração automática de cláusulas e a indentação impecável.

### 2. 🛡️ Radar Analítico de Vulnerabilidades
Escaneamento semântico em tempo real de termos perigosos, ambíguos ou prejudiciais. O motor de IA analisa o documento continuamente e gera um **Score de Segurança dinâmico (0-100%)** com badges responsivos que mudam de cor conforme a classificação do risco (Excelente, Moderado, Crítico).

### 3. 🌳 Mapa do Instrumento (Árvore de Blueprint)
Uma visualização lateral da hierarquia de cláusulas e parágrafos estruturada em forma de blueprint técnico. Permite navegação fluida em tempo real, indicando instantaneamente o status de auditoria de cada cláusula com um scroll minimalista e elegante para manter a densidade do painel.

### 4. ⚡ Sincronia Dual-Sync Sem F5
Colaboração com zero latência. Sincronização em tempo real na biblioteca lateral e nos workspaces baseada no protocolo de broadcast do **Supabase Realtime** combinado com o controle de concorrência **Yjs**. Qualquer alteração de participantes é refletida instantaneamente sem necessidade de atualizar a página.

### 5. 🕸️ Rede de Conexões Estratégicas
Um grafo interativo bidimensional alimentado por **ForceGraph2D** que mapeia visualmente relações contratuais, partes assinantes, contratos colaterais e oportunidades de mercado em tempo real.

### 6. 🔒 Assinatura Criptográfica e Biometria
Blindagem jurídica por meio de selos digitais de hash criptográfico gerados localmente e verificação biométrica para garantir a inviolabilidade das minutas assinadas.

## 💻 Tech Stack e Arquitetura

* **Core Framework:** Next.js 16.2.6 (Turbopack) com TypeScript rígido.
* **Component System:** Shadcn UI altamente customizado sob tokens de design Dark Occult.
* **Text Editor Engine:** TipTap Editor com extensões personalizadas para numeração de cláusulas e indentação automatizada.
* **Database & Realtime:** Supabase (Auth, Realtime Broadcast, Storage).
* **Styling:** Tailwind CSS (sem utilização de estilos inline; coerência semântica pura).
* **Visual Data:** ForceGraph2D para mapeamento relacional.

## 🚀 Como Iniciar

1. Clone o repositório e configure as variáveis de ambiente em um arquivo `.env` na raiz:
   ```env
   NEXT_PUBLIC_SUPABASE_URL=seu-url-supabase
   NEXT_PUBLIC_SUPABASE_ANON_KEY=sua-chave-anon-supabase
   ```

2. Instale as dependências:
   ```bash
   npm install
   ```

3. Suba o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```
